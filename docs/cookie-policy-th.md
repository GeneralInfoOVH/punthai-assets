ปรับปรุงล่าสุด: 19 มกราคม 2566  
1\. คุกกี้คืออะไร  
คุกกี้ คือไฟล์ขนาดเล็กที่อาจถูกติดตั้งและ/หรือจัดเก็บไว้ในอุปกรณ์ของคุณ (เช่น คอมพิวเตอร์ โทรศัพท์มือถือ หรือแท็บเล็ต) เมื่อคุณเข้าชมเว็บไซต์  
คุกกี้เหล่านี้จะถูกส่งกลับไปยังเว็บไซต์ต้นทางในแต่ละครั้งที่คุณเข้าชม หรือส่งไปยังเว็บไซต์อื่นที่จดจำคุกกี้เดียวกันนั้นได้  
คุกกี้มีวัตถุประสงค์การใช้งานที่หลากหลาย รวมถึงช่วยให้เว็บไซต์ทำงานได้อย่างมีประสิทธิภาพ เพิ่มประสิทธิภาพการทำงาน และเก็บรวบรวมหรือติดตามข้อมูลส่วนบุคคลเพื่อการปรับปรุงเว็บไซต์และการตลาดแบบเจาะจงกลุ่มเป้าหมาย  
2\. ประเภทของคุกกี้ที่เราใช้  
คุกกี้ที่ใช้บนเว็บไซต์นี้ถูกจัดประเภทตามการใช้งาน ดังนี้:  
คุกกี้ที่จำเป็นอย่างยิ่ง (Strictly Necessary Cookies)  
คุกกี้เหล่านี้มีความจำเป็นอย่างยิ่งเพื่อให้เว็บไซต์ทำงานได้อย่างถูกต้อง และช่วยให้คุณสามารถใช้งานฟีเจอร์ต่างๆ บนเว็บไซต์ได้  
ตารางดังต่อไปนี้:

| ประเภทคุกกี้ | ชื่อ | วัตถุประสงค์ | ระยะเวลาจัดเก็บ | ผู้ให้บริการ |
| :---- | :---- | :---- | :---- | :---- |
| จำเป็นอย่างยิ่ง | Phpse Ssid | ใช้เพื่อจัดเก็บและแยกแยะ Session ID เฉพาะของคุณสำหรับการจัดการเซสชันในการเข้าชมแต่ละครั้ง | ลบเมื่อปิดเบราว์เซอร์ | เว็บไซต์ |
| จำเป็นอย่างยิ่ง | ASP.NET Session ID | ใช้เพื่อนำ Session ID ไปใช้ในการดำเนินงานที่เกี่ยวข้องกับโปรแกรม | ลบเมื่อปิดเบราว์เซอร์ | เว็บไซต์ |

3\. การจัดการคุกกี้  
คุณสามารถจัดการการใช้งานคุกกี้ได้ตลอดเวลาโดยการคลิกที่ไอคอนการตั้งค่าบนเว็บไซต์  
คุณสามารถสลับการตั้งค่าระหว่าง 'ยอมรับ' และ 'ปฏิเสธ' จากนั้นคลิก 'บันทึกและปิด'  
คุณอาจจำเป็นต้อง 'รีเฟรช' หน้าจอเพื่อให้การตั้งค่าเหล่านี้มีผล  
นอกจากนี้ คุณสามารถจัดการคุกกี้ผ่านการตั้งค่าของเว็บเบราว์เซอร์หรืออุปกรณ์มือถือของคุณได้ รวมถึง:

* Google Chrome  
* Microsoft Edge  
* Mozilla Firefox  
* Microsoft Internet Explorer  
* Opera

4\. การปรับปรุงนโยบายคุกกี้  
นโยบายคุกกี้นี้อาจมีการแก้ไขหรือปรับปรุงเป็นระยะ เพื่อให้สอดคล้องกับแนวปฏิบัติที่ดีที่สุด กฎหมาย ข้อบังคับ หรือข้อกำหนดอื่น ๆ ที่เกี่ยวข้อง  
บริษัทขอแนะนำให้คุณตรวจสอบการปรับปรุงเหล่านี้เป็นระยะ
